Hexo错误集锦

Cannot　read　property　‘offset’　of　null

原因: _config.yml下timezone配置错误。

解决方法: 将timezone配置成时区名称。

例如: timezone:Asia/Shanghai

Node

socket.io

安装

npm install socketio

下载源码：https://github.com/Automattic/socket.io。
访问管网：http://socket.io。

服务端

注册事件

var io = require('socket.io');  //引用模块
var io = require('socket.io')(80);  //监听端口
io.on('connection',function (socket) {
  //注册连接成功事件
  socket.on('disconnect',function () {
     //注册断开连接事件
  });
});

启动服务(与express配合)

var express = require('express');
var app = express();
app.get('/',function (req,res) {
  res.status('200').send('message');
});
var server = require('http').createServer(app);
var io = require('socket.io')(server);
io.on('connection',function(socket) {
      //do something
});
server.listen(3000);

客户端

服务端运行后，会自动在根目录下生成客户端的js文件，客户端通过固定路径/socket.io/socket.io.js添加引用

<script src='/socket.io/socket.io.js'></script>
